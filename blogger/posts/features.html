Everything Claude Bridge can do — organized by category. All features work from your Telegram chat.

Core Communication

Text Messages — Send any message and Claude Code processes it as a prompt. Full multi-turn conversation with session persistence.

Image Analysis — Send photos (with or without captions). Claude sees the image and responds — screenshot debugging, UI review, document scanning.

Voice Conversation — Send a voice message, it's transcribed locally via Whisper, Claude responds with both text and spoken audio.

Document Handling — Send PDF, code files, or logs directly. Claude reads and analyzes the content without you needing to describe it.

Streaming Progress — See what Claude is doing in real-time: which files it's reading, what commands it's running, what it's editing. No more staring at a blank screen.

Voice System

/voice — Voice Settings — Toggle voice replies on/off, switch between TTS engines, browse and preview voices before selecting.

Two TTS Engines

  • edge-tts — Free, 8 Chinese voices including Liaoning and Shaanxi dialects
  • ElevenLabs — Cloud TTS with 21 premium voices, natural intonation

/el — ElevenLabs Dashboard — Check remaining credits, monthly usage, voice slots, and next billing date — all from Telegram.

Project Management

/p — Switch Projects — Tap to switch between registered projects. Each project has its own CLAUDE.md, session state, and cost tracking. InlineKeyboard for quick selection.

/p add / /p rm — Register or remove projects on the fly.

/model — Model Selection — Switch between Opus (deep reasoning) and Sonnet (fast responses).

/effort — Thinking Depth — Set Low / Medium / High to control how deeply Claude thinks before responding.

/think — Deep Mode — One tap to switch to Opus + High effort for complex tasks.

Task Orchestration & Automation

/task — Two-Phase Execution — For operations that modify your system: Phase 1 analyzes in readonly mode and presents a plan. You review and confirm. Phase 2 executes with full tool access. Safe by design.

/cron — Scheduled Tasks — Register prompts that run on a schedule. Add, list, pause, resume, or remove recurring jobs — health checks, reports, deployments, anything.

Cost Control

/budget — Daily Budget — Set a spending cap. Toggle on/off, adjust the amount. Persisted in SQLite, takes effect immediately without restart.

/cost — Cost Reports — View spending: today, last 7 days, broken down by project. Know exactly where your budget goes.

Smart Model Routing — Short, simple queries (under 80 characters, non-complex) automatically use Sonnet + Low effort. Complex tasks stay on Opus. Saves cost and responds faster — no manual switching needed.

Security

User Whitelist — Only Telegram user IDs in allowFrom can interact with the bot. Everyone else is silently ignored.

Keychain Integration — Bot token stored in macOS Keychain, not plaintext config. Referenced via !security prefix in config.json.

Sensitive Data Masking — Passwords, tokens, and API keys in your messages are automatically masked in Claude's responses.

Auto-Delete Sensitive Messages — Messages containing passwords or tokens are automatically deleted from the chat after processing.

Operations

/restart — Remote Restart — Restart the Claude Bridge service from Telegram. LaunchAgent automatically respawns the process.

/new — New Session — Reset the current conversation. Start fresh without restarting the service.

/status — Status Dashboard — Current project, model, effort level, session info, and all available actions in one view.

LaunchAgent — Runs as a macOS background service. Starts on boot, restarts on crash. Zero maintenance once configured.

Per-User FIFO Lock — Each user's requests are queued in order. A global semaphore limits concurrent workers to prevent resource exhaustion.
